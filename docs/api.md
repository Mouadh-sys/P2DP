# P2DP API

## Authentication

- `POST /api/auth/register`
- `POST /api/auth/token`

## Projects

- `POST /api/projects`
- `GET /api/projects`
- `GET /api/projects/{id}`
- `DELETE /api/projects/{id}`
- `POST /api/projects/{id}/environments`
- `GET /api/projects/{id}/environments`

## Environments

- `GET /api/environments/{id}`
- `POST /api/environments/{id}/upload`
